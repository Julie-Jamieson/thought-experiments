# NetLogo Testing

This folder contains the NetLogo-based testing framework for CG-AI: The Companion-Guardian Project.

The purpose of this stage is to move the early conceptual simulations into an agent-based modelling environment, where stewardship anchors, behavioural rules, and decision outcomes can be tested with clearer structure.

These tests should be treated as V2 simulation work.

The original ChatGPT-based simulations are preliminary and exploratory. NetLogo testing is intended to strengthen the model, clarify assumptions, and make the experiment easier to review, repeat, and critique.
