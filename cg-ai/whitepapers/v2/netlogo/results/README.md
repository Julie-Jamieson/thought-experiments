# Results

This folder contains preliminary simulation observations and run logs for CG-AI NetLogo experiments.

The simulations within this directory are exploratory and intentionally simplified. Their purpose is to examine emergent behavioural patterns under controlled rule sets and environmental conditions.

These models are not predictive representations of real-world societies, economies, ecosystems, or intelligence systems.

Each test may include:
- hypothesis
- simulation parameters
- behavioural rules
- observed outcomes
- limitations
- future adjustments

The purpose of these experiments is to iteratively explore how differing incentive structures, extraction behaviours, regenerative systems, and cooperative dynamics influence long-term stability within simplified agent-based environments.
