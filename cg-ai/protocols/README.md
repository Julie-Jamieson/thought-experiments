# Protocols

This section contains structured experimental protocols, behavioural conditions, simulation rules, and methodological frameworks related to CG-AI.

Protocols are intended to formalize exploratory experiments into repeatable and analyzable structures.

These documents may include:

* experimental setup
* behavioural constraints
* resource conditions
* agent interaction rules
* communication permissions
* memory conditions
* scoring or evaluation criteria
* termination conditions
* simulation assumptions
* known limitations

The purpose of this section is to separate conceptual exploration from structured experimentation.

Protocols within this repository are exploratory in nature and are not presented as validated scientific methodologies or peer-reviewed research frameworks.

As the project evolves, protocols may continue to change, expand, or branch into revised experimental versions.
