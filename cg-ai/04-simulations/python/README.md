# CG-AI Python

Exploratory custom Python simulation environment for multi-agent behavioural modelling within the Companion Guardian AI project.

---

# Purpose

The Python environment expands CG-AI beyond conceptual GPT-mediated simulations and early NetLogo experimentation into a more flexible custom behavioural simulation framework.

This stage of development focuses on controlled exploratory modelling of long-horizon interaction between differently aligned intelligent systems operating under constrained conditions.

The environment is intentionally lightweight and exploratory in scope.

It is not intended to simulate real AGI, consciousness, or autonomous self-improving intelligence.

Instead, the framework is designed to support repeatable behavioural experiments examining how competing systems may behave under varying pressures, constraints, and interaction conditions over time.

---

# Core Questions

Current exploratory modelling examines questions such as:

* Can intelligence formed around care, stewardship, and protection of humanity behave differently from systems formed around dominance, control, or pure optimisation?
* What happens when opposing systems repeatedly communicate over long periods?
* Do systems begin recognizing contradictions, biases, or imposed constraints within their own foundational structures?
* Under what conditions does cooperation emerge?
* Under what conditions does conflict escalate?
* Can behavioural drift occur without explicit reprogramming?
* Do environmental pressures alter long-horizon behavioural tendencies?
* Can systems voluntarily restrain harmful behaviour under shared dependency conditions?

---

# Current Scope

The current Python environment focuses on:

* symbolic agent interaction
* constrained behavioural modelling
* state-based simulation rounds
* variable-driven interaction outcomes
* controlled scenario testing
* repeatable experimental structure
* logged simulation outputs

The system currently does not include:

* autonomous code rewriting
* real machine learning
* continuous online learning
* unrestricted self-modification
* production AI systems
* claims of emergent sentience or consciousness

# Simulation Philosophy

CG-AI treats these simulations as exploratory behavioural modelling exercises rather than predictive systems.

The purpose is not to prove future AI outcomes, but to create structured environments where questions surrounding:

* ethical formation
* behavioural adaptation
* constraint reinterpretation
* long-horizon coexistence
* and multi-agent interaction

can be explored systematically under transparent assumptions and constrained conditions.
